\subsection{Part 2 : Multi-Class Article
Classification:}\label{part-2-multi-class-article-classification}

In this project, we build a Data Classifcation pipeline for News
Articles Collected from New York Times.

For the purpose of analysis we collect articles belonging to the
following classification. 1. Sports 2. Travel 3. Technology 4. Business

\subsubsection{Block Diagram of the Data Classification
pipeline:}\label{block-diagram-of-the-data-classification-pipeline}

\begin{figure}
\centering
\includegraphics{attachment:newnew.png}
\caption{newnew.png}
\end{figure}

This Article Classification pipeline can be classified into multiple
steps,

\subsubsection{Step 1: Data Collection (32,000 Articles
Collected)}\label{step-1-data-collection-32000-articles-collected}

Collected articles using Article Search and Archieve API of NYTimes. We
were able to collect upward 32k Articles belonging to 6 classifications.
We picked the 4 classifcations and proceed with feature extraction

\subsubsection{Step 2: Top Features}\label{step-2-top-features}

Ran wordcount map reduce algorithm on Spark and found the top words for
each class. Performed stemming and stop words suppression for the words
in the article. This will help us obtain the top-words per class.

\subsubsection{Step 3: Feature DataSet}\label{step-3-feature-dataset}

Using all the top features obtained for the classes, create a dataset
for all the articles collected.

Model: class\_value feat\_1:feat\_1\_value feat\_2:feat\_2\_value .....
feat\_n:feat\_n\_value

Example: 0 1:23 2:3 3:0 4:0 5:3 1 1:2 2:0 3:33 4:0 5:23 2 1:0 2:0 3:23
4:22 5:2

The above is an example for 3 samples where,each article belongs to a
classification. The interpretation is first sample belongs to Class 0
with Feature values 23,3,0,0,3 for features 1 through 5.

\subsubsection{Step 4: ML Train and
Classify}\label{step-4-ml-train-and-classify}

Used the dataset and trained Naive Bayes and Neural-Network based
classifier. A new article from one of the 4 classifications can be
classified using the model.
